
# Restaurant CRM Dashboard

A modern CRM dashboard built by ForgeSOL to help restaurants manage customers, orders, and business performance.

## Features

* Customer Management
* Order Tracking
* Revenue Analytics
* Business Dashboard
* Customer Tags (VIP, Regular, New)

## Tech Stack

* Next.js
* React
* Tailwind CSS
* PostgreSQL

## Purpose

This project is part of ForgeSOL's portfolio and demonstrates how restaurants can streamline operations through custom software.
